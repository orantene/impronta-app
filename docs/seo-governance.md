# SEO governance (Phase 8.5)

## Slugs

- Uniqueness per **locale** (or documented composite rule).
- **Published** slug changes require **redirect** rows + staff warning.

## Metadata

- Per-page/post overrides + **global defaults** (site name suffix, default description, default OG).
- Talent profile URLs may stay app-generated; sitemap rules documented when CMS merges.

## Robots / sitemap

- `noindex`, `include_in_sitemap` per content row where applicable.
- Utility routes (login, dashboard) excluded by app config.
