Tulala · Builder experience · Design spec v3

A complete canvas builder — top-bar mission control, full design control, drafts and revisions, every flexibility a real prototype can demand.

v3 raises the ambition. The top bar becomes mission control — pages, save/publish split, page settings, revisions, preview, share. The inspector gains real design control across five tabs (Content, Layout, Style, Responsive, Animation) — every spacing, sizing, container, breakpoint, color, type, background, border, shadow, visibility and animation property a top-tier builder offers, so any imported HTML/coded prototype can be modeled cleanly. Page settings, revisions, and the publish drawer all use the same drawer chrome — same header structure, same tools (expand-cycle / fullscreen / close), same paper-tinted body with white card modules, same input system. One visual language, applied everywhere.

1Top bar — mission control

Today's top bar has Editing/Draft/undo/redo/viewport/Publish/Exit. Real builders give the operator far more global access without cluttering. v3 adds: page picker (multi-page), Save/Publish split (Save draft vs. Publish live), Page settings, Revisions, Preview, Share, Help — each as a discrete, well-labeled control.

3Style — every visual property

Background (color/gradient/image/video), border (width/style/color/radius per corner), shadow (or stack of shadows), typography overrides (family/weight/size/line-height/ letter-spacing/color/transform/decoration), opacity, blend mode, filter (saturate/ brightness/contrast/blur). Imported HTML can specify any of these — the panel reads them back and lets the operator tweak.
