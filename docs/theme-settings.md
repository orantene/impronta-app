# Theme settings (Phase 8.5–8.6)

## Allowed (token-based only)

Aligned with plan hard rule:

- Primary / accent color  
- Button style variant  
- Border radius scale  
- Logo (light/dark)  
- Default OG image  

## Storage

Prefer `public.settings` keys documented in decision-log; no freeform CSS or per-page color overrides.

## Not allowed

Per-page theme, typography override, spacing/grid control — see frozen plan hard rule.
